## 1. Premise

Worlds differ not only in art direction but in **Observation Mode** — the policy that decides which slice of \(\mathbb{R}^{4}\) becomes host-visible. Directors treat Modes as lenses with rules, not as Instagram filters.

## 2. World types (declared creative kit)

| World id | Feel | Observation emphasis | Avoid |
| --- | --- | --- | --- |
| W-SLICE | Thin luminous membrane | Orthographic W cut | Fake “portal goo” without policy |
| W-DRIFT | Slow parallax of stacked slices | Time-varying Mode | Claiming realtime 4–6 ms |
| W-FOLD | Geometric crease / hyper-box reveal | Accel structure as set dressing | “World’s first engine” title cards |
| W-HOST | Austere 3D landing bay | Projection + lineage visible as HUD glyphs | “Seamless Unreal” supers |

## 3. Visual grammar

- **Axes:** Prefer subtle W tick marks over noisy HUD spam.  
- **Accel poetry:** Hyper-box edges may appear as scaffolding — labeled **schematic** if shown.  
- **Projection beat:** Always show a clear “4D → 3D” handoff once per piece.  
- **Legend:** If diagrams appear, keep declared/roadmap colors from [`ENGINE_ARCHITECTURE_DIAGRAM.md`](./ENGINE_ARCHITECTURE_DIAGRAM.md).

## 4. Sound / silence

Sparse tones keyed to Mode changes. Silence is allowed when a Mode locks. Score direction: **TBD**.

## 5. Continuity rules

1. Same Mode id ⇒ same optical rules across shots.  
2. Lineage glyphs (if shown) must not contradict PLP vocabulary.  
3. Host engine plates only from **existing** runnable paths — captioned as partial/prior when used.  
4. No invented Nanite/Lumen 4D hero shots.

## 6. Character / camera notes (optional fiction)

Protagonist is an *Observer* — someone who changes Modes deliberately. Villain trope (optional): an engine that flattens W without lineage. Keep fiction clearly separated from product capability claims in any press cut.
