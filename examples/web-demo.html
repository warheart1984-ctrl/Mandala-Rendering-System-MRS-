Controls

Drag to scrub θ / camera Y · wheel to zoom · Space pause · R reset. Animation is parametric cinematicRotation, not a governed timeline player.

Declared / not wired in this demo

  • WebGPU bloom, tone map, chromatic aberration (PostProcessor)
  • Shadow maps / env reflections (ShadowMapper, EnvironmentMapper)
  • LOD managers (API present; not bound to this canvas loop)
  • Microphone / audio-reactive mapping (browser AudioAnalyzer exists; not enabled here)
  • Browser GLTF/OBJ export (Node ExportManager only)
  • Governed host timelines — use root index.html + npm start

Serve the repo root over HTTP (ES modules). From root: npm run serve then open /examples/web-demo.html. Gallery · tutorials · maturity notes: examples/README.md.
